FAQ SECTION:

Container: py-20, max-w-4xl mx-auto px-6

Header: "AI Agent FAQs"

12 Questions (Accordion):

Q1: "What's the difference between AI agents and chatbots?"
A: "Chatbots are reactive—they respond to user questions. AI agents are proactive—they autonomously execute tasks without waiting for input. Chatbots chat; agents act. Example: A chatbot answers 'What's my order status?' An agent checks order status daily and alerts you if there's a delay."

Q2: "How do AI agents make decisions?"
A: "Agents use large language models (GPT-4, Claude) to reason through problems. Given a goal like 'Qualify this lead,' the agent breaks it down: (1) Research company, (2) Analyze fit, (3) Score priority, (4) Route to sales or nurture. The AI plans steps, executes using tools (web scraping, CRM API), and decides next actions based on results."

Q3: "Can agents make mistakes?"
A: "Yes, which is why we build guardrails. Agents operate with constraints (e.g., 'Never send emails without approval for high-priority issues'), log all actions for review, and escalate uncertain decisions to humans. We aim for 90-95% accuracy, with human oversight for critical tasks."

Q4: "What tools can AI agents use?"
A: "Hundreds of tools via APIs: Email (Gmail, Outlook), CRM (HubSpot, Salesforce), databases (PostgreSQL, MongoDB), calendars, Slack, Google Sheets, web scraping, payment systems, analytics platforms. If it has an API or web interface, agents can likely use it."

Q5: "How long does it take to build an AI agent?"
A: "Simple single-task agent: 3-4 weeks. Multi-step workflow agent: 4-6 weeks. Multi-agent system: 6-10 weeks. Timeline depends on workflow complexity, number of integrations, and testing requirements."

Q6: "Do agents work 24/7?"
A: "Yes! Agents run continuously on cloud servers. They can process emails at 3 AM, run daily reports at 7 AM, monitor competitors hourly—all without human intervention. You set the schedule and triggers."

Q7: "What happens if an agent encounters an error?"
A: "Agents are built with error handling: (1) Retry failed actions (e.g., if API is down, try again in 5 min), (2) Log errors for debugging, (3) Escalate to humans if repeated failures, (4) Send alerts via email/Slack. They don't fail silently."

Q8: "Can we supervise what agents do?"
A: "Absolutely! We provide monitoring dashboards showing: Every action taken, Decisions made, Errors encountered, Time saved. You can set approval requirements for sensitive actions (e.g., 'Send email only after human approves')."

Q9: "Can agents learn from experience?"
A: "Yes, in several ways: (1) We retrain agents on new data periodically, (2) Agents use RAG (Retrieval-Augmented Generation) to learn from your docs, (3) Human feedback improves future performance, (4) Agents maintain memory of past actions for context."

Q10: "What about security and data privacy?"
A: "Agents access only what you authorize (via API keys, OAuth). Data is encrypted in transit and at rest. We can deploy agents on your infrastructure for full control. Compliance: GDPR-friendly, can be HIPAA-compliant if needed. No training on your data without permission."

Q11: "Can agents replace entire roles?"
A: "Agents replace tasks, not people. They handle repetitive, time-consuming work (reading emails, data entry, research), freeing humans for strategic, creative, and relationship work. Best use: Augment your team, not replace it. Example: Agent processes 80% of support tickets, humans handle the nuanced 20%."

Q12: "What are monthly costs to run an agent?"
A: "AI API costs: £200-800/month (depends on volume—GPT-4 is more expensive than GPT-3.5). Hosting: £100-200/month. Monitoring: Included. Example: Email agent processing 200 emails/day costs ~£400/month total. Much cheaper than hiring a full-time employee (£3,000+/month)."
