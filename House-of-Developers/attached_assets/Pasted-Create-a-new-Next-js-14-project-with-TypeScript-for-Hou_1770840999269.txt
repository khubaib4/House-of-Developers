Create a new Next.js 14 project with TypeScript for "House of Developers" - a premium UK-based digital product development agency.

PROJECT SETUP:

1. Initialize Next.js 14 with App Router and TypeScript
2. Install these dependencies:
   - tailwindcss
   - @shadcn/ui (button, card, dialog components)
   - next-themes
   - framer-motion
   - lucide-react
   - @next/font

3. Configure Tailwind CSS with this custom theme:

Colors:
Light Mode:
  primary: '#0066FF'
  background: '#FFFFFF'
  foreground: '#0A0A0A'
  secondary: '#F8F9FA'
  accent: '#00C853'
  border: '#E5E7EB'
  muted: '#F3F4F6'

Dark Mode:
  primary: '#3B82F6'
  background: '#0A0A0A'
  card: '#1A1A1A'
  foreground: '#F5F5F5'
  secondary: '#1F1F1F'
  border: '#2A2A2A'
  muted: '#2A2A2A'

Typography:
  - Font family: Inter (from Google Fonts)
  - H1: text-5xl font-bold (mobile: text-4xl)
  - H2: text-4xl font-bold (mobile: text-3xl)
  - H3: text-2xl font-semibold (mobile: text-xl)
  - Body: text-base leading-relaxed

4. Set up next-themes provider for dark/light mode switching

5. Create this folder structure:

       /ServicesGrid.tsx

       /ProjectsShowcase.tsx
       /TechStack.tsx
       /CTASection.tsx

     /utils.ts
