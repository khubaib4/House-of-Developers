Framer Motion:
- Container: fade-in and slide-up on scroll

  * transition: duration 0.8, ease: "easeOut"
  * viewport: once: true, margin: "-100px"

- Mockup: parallax effect on scroll
  * Subtle movement (3-5px) on scroll
  * Scale on hover: 1.02
  * Transition: smooth 300ms

- Feature items: stagger animation
  * Each item appears with 100ms delay
  * Slide from left with opacity

Hover Effects:
- Mockup container: lift effect (translateY: -4px)
- Shadow enhancement on hover
- Smooth transitions

3D Tilt Effect (optional, subtle):
- On mouse move, mockup tilts slightly (5-10 degrees max)
- Smooth spring animation
- Only on desktop

Glassmorphism:
- Mockup background option: backdrop-blur-xl
- Semi-transparent backgrounds
- Border: 1px solid rgba(255,255,255,0.1)

RESPONSIVE BEHAVIOR:

- Reduced padding
- Smaller mockup sizes

Desktop:
- Side-by-side layout
- Mockup fills available space
- All animations enabled

ACCESSIBILITY:
- Semantic HTML (section, article tags)
- ARIA labels for decorative elements
- Keyboard focus states
- Reduced motion support (prefers-reduced-motion)

PART 4: DESIGN SPECIFICATIONS

MOCKUP DESIGN GUIDELINES:

Color Palette for Mockups:
- Primary actions: Use theme primary color
- Success states: #00C853 (green)
- Warning states: #FFA726 (orange)
- Error states: #EF5350 (red)
- Neutral backgrounds: theme muted colors
- Text: theme foreground colors

Typography in Mockups:
- UI labels: text-xs or text-sm
- Numbers/metrics: text-2xl to text-4xl, font-bold
- Descriptions: text-sm, text-muted-foreground

Spacing:
- Consistent padding: p-4, p-6, p-8
- Gaps between elements: gap-4, gap-6
- Card spacing: space-y-4

Charts/Graphs:
- Use CSS gradients for backgrounds
- Simple line/bar representations with divs
- Animated on scroll (optional)
- Color: primary or accent gradients

Data Tables:
- Clean borders (border-b)
- Alternating row backgrounds (even:bg-muted/50)
- Sticky headers
- Proper alignment

Icons:
- Consistent size (16-24px)
- Primary color for active states
- Muted for inactive

Shadows & Depth:
- Cards: shadow-sm
- Elevated elements: shadow-md
- Mockup frames: shadow-2xl
- Hover states: shadow-lg

Animations:
- Subtle pulse on live indicators
- Fade-in for charts
- Typing animation for chat
- Smooth transitions (200-300ms)

DARK/LIGHT MODE:
- All mockup content must support both modes
- Use theme colors consistently
- Adjust contrast for readability
- Test in both modes

PERFORMANCE:
- Lazy load mockup components
- Use CSS transforms for animations (GPU accelerated)
- Optimize SVGs if used
- Minimize re-renders

ACCESSIBILITY:
- All mockup content is decorative (aria-hidden="true")
- Actual information is in the text content (features list)
- Proper color contrast
- Animations respect prefers-reduced-motion
