ANIMATIONS & INTERACTIONS:

Framer Motion Variants:

Mega Menu Container:
- initial: { opacity: 0, y: -20 }
- animate: { opacity: 1, y: 0 }
- exit: { opacity: 0, y: -10 }
- transition: { duration: 0.2, ease: "easeOut" }

Menu Items (stagger):
- Container: staggerChildren with 0.03s delay
- Children: fade and slide from left
  * initial: { opacity: 0, x: -10 }
  * animate: { opacity: 1, x: 0 }

Featured Cards:
- Hover: scale 1.02, border color change
- Transition: 200ms ease

Hover States:
- All clickable items: hover:bg-muted
- Icons: group-hover color change to primary
- Smooth transitions (200-300ms)

---

ACCESSIBILITY:

Keyboard Navigation:
- Tab through all menu items
- Enter/Space to activate links
- Escape to close menu
- Focus trap when menu is open
- Focus first item when opening
- Return focus to trigger when closing

ARIA Attributes:
- Menu trigger: aria-expanded, aria-haspopup="true"
- Menu container: role="menu"
- Menu items: role="menuitem"
- Proper aria-labels for all links

Screen Readers:
- Announce when menu opens/closes
- Descriptive labels for all items

RESPONSIVE BEHAVIOR:

Desktop (≥ 1024px):
- Full mega menu with 3 columns
- Hover to open (with delay)
- Click to toggle

Tablet (768px - 1023px):
- Simplified 2-column layout
- Remove featured cards or stack them
- Adjust spacing

Mobile (< 768px):
- No mega menu
- Use accordion in mobile hamburger menu
- Simple nested list of services
- Full-screen mobile menu overlay

DARK/LIGHT MODE:

Light Mode:
- Background: white with subtle backdrop-blur
- Borders: gray-200
- Hover: gray-100
- Featured cards: gray-50

Dark Mode:
- Background: #1a1a1a with backdrop-blur
- Borders: gray-800
- Hover: #2a2a2a
- Featured cards: #222

Ensure all text has proper contrast in both modes

PERFORMANCE:

- Lazy render mega menu content (only render when opened first time)
- Prefetch linked pages on hover
- Optimize icons (use lucide-react treeshaking)
- Minimal re-renders
- Smooth 60fps animations

---

CLOSE BEHAVIOR:

Mega menu should close when:
- User clicks outside menu
- User hovers away for 300ms
- User presses Escape key
- User clicks a menu item (navigate away)
- User scrolls down significantly (optional)

Mega menu should NOT close when:
- User hovers within menu area
- User is tabbing through items
- Mouse briefly passes through gap between trigger and menu
