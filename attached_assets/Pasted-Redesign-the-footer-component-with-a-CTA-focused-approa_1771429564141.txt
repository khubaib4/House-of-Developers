DESIGN SPECIFICATIONS:

Theme-Adaptive Colors:
* Use CSS variables for automatic dark/light switching
* bg-card (adapts to theme)
* text-foreground and text-muted-foreground
* border (theme-aware)
* CTA gradient uses primary color (works in both modes)

CTA Banner:
* Always uses primary gradient (stands out in both themes)
* White text on colored background
* Prominent, converts visitors before they leave

Typography:
* Footer headings: text-sm font-semibold uppercase 
  tracking-wide
* Links: text-sm
* Copyright: text-sm
* CTA heading: text-3xl md:text-4xl

Spacing:
* CTA banner: py-12 md:py-16
* Main footer: py-16
* Bottom bar: py-6
* Consistent gaps: gap-4, gap-8, gap-12

Hover States:
* Links: hover:text-primary
* Social icons: hover:bg-primary hover:text-white
* Buttons: hover effects with scale or opacity
* Smooth transitions (200ms)

Mobile Responsive:
* CTA: Stack vertically on mobile, side-by-side on desktop
* Footer columns: 1 col mobile → 2 cols tablet → 4 cols desktop
* Logo section: Full width mobile, left column desktop
* Center-align on mobile, left-align on desktop

Accessibility:
* All links keyboard-accessible
* Proper ARIA labels for social icons
* Semantic HTML (<footer>, <nav>)
* Sufficient color contrast in both themes
