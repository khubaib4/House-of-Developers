SEO BEST PRACTICES TO FOLLOW:

1. TITLES (50-60 characters):
   - Include primary keyword
   - Include location if relevant (UK, London, etc.)
   - Include brand name at end
   - Format: "Service | Benefit | Brand"
   - Make unique for every page

2. DESCRIPTIONS (150-160 characters):
   - Include primary and secondary keywords naturally
   - Include a call-to-action
   - Mention key benefits or USPs
   - Make it compelling and click-worthy
   - Unique for every page

3. OPEN GRAPH:
   - og:title can match title or be slightly different
   - og:description should match meta description
   - og:image should be 1200x630px (create if missing)
   - og:type: "website" for pages, "article" for blog posts
   - og:url: Full canonical URL

4. CANONICAL URLs:
   - Set on every page to prevent duplicate content issues
   - Use full URL including https://

5. STRUCTURED DATA:
   - Use JSON-LD format (in <script> tag)
   - Validate with Google Rich Results Test
   - Include on all major page types
